# v2 UI screenshots

Preview of the new responsive web UI on branch
`claude/thermostat-redesign-plan-P7XpY`. All screenshots were captured by
rendering the templates with **mocked thermostat data** in headless Chromium
(Playwright), not from a live device.

Mock state: target 19.5 °C, room 19.2 °C, outside 8.4 °C, scheduled mode,
HW 5 / FW 30050046, plus a sample standard week and two demo agenda
exceptions on Wed (Comfort 09:00–17:00) and Thu (Away 19:30–23:00) of the
current week.
